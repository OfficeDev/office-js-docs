# ChartDataLabel Object (JavaScript API for Excel)

Represents the data label of a chart point.

## Properties

| Property	   | Type	|Description| Req. Set|
|:---------------|:--------|:----------|:----|
|position|string|DataLabelPosition value that represents the position of the data label. Possible values are: None, Center, InsideEnd, InsideBase, OutsideEnd, Left, Right, Top, Bottom, BestFit, Callout.|[TBD](../requirement-sets/excel-api-requirement-sets.md)|
